Cryptic Crossword Evaluation Benchmark

Evaluating Large Language Models on Cryptic Crossword Solving

About This Benchmark

Cryptic crosswords represent a unique challenge for language models, requiring sophisticated wordplay understanding, lateral thinking, and knowledge of cryptic clue conventions. Unlike standard crosswords that test factual recall, cryptic clues involve anagrams, hidden words, double definitions, and other linguistic puzzles.

This benchmark evaluates LLM performance on cryptic crossword clues extracted from real puzzles. Each model is presented with a clue and the answer length, and must produce the correct answer. Accuracy is measured as exact match (case-insensitive, alphanumeric only).

Methodology

  • Dataset: Cryptic crossword clues from published puzzles
  • Prompt: Each clue is presented with the answer length hint
  • Scoring: Exact match after normalisation (case-insensitive, alphanumeric only)
  • Error bars: Standard error calculated as sqrt(p*(1-p)/n)
